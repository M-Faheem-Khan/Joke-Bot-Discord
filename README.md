# Joke-Bot-Discord
Joke Bot for Discord

### Commands
- `!joke` - will get you a new joke

### Built Using
- https://geek-jokes.sameerkumar.website/api - api for jokes
- NodeJS
- Node-Fetch
- Discord.js
